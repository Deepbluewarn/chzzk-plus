## Feature

1.  PIP 모드 버튼
2.  방송 미리보기
3.  방송 화면 캡처저장
4.  생방송 녹화
5.  치즈 제거 모드
6.  채팅 색상 모드
7.  채팅 크기 조절
8.  방송 단축키
9.  즐겨찾기 (팔로우 채널 중 우선순위 올리기)
10. 생방송 빨리감기
11. 오디오 압축
